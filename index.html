Plataforma web con React.js y Spring Boot que permite crear y organizar armarios virtuales. Integra Spring Security para autenticación, desplegada en Azure, desarrollada con metodologías ágiles Scrum y Kanban.

Tecnologías: React.js, Spring Boot, Spring Security, Azure, Scrum, Kanban
